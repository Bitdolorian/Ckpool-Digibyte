# Ckpool-Digibyte — Full DigiByte Mining Stack

This repository contains a complete, source-based DigiByte mining stack:

- DigiByte Core (digibyte-core/)
- CKPool-DGB (ckpool-source/)
- CKStats dashboard (ckstats/)
- Example configs (configs/)
- Build + start scripts (scripts/)

## Build everything
./scripts/build-all.sh

## Start everything
./scripts/start-all.sh

## Stop everything
./scripts/stop-all.sh

## Notes
- Do NOT commit blockchain data.
- .gitignore protects runtime artifacts.
- All configs are in configs/.
